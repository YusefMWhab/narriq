Welcome to NarrIQ

Your comprehensive guide to understanding and using the smart market research analysis system.

NarrIQ is engineered to empower market researchers and data analysts to decode open-ended textual responses at lightning speed with maximum precision. The system converts thousands of raw, unstructured verbatim comments into a **unified, thoroughly calculated Codebook** that is easy to interpret and filter.

Project Setup & Data Ingestion Modes

NarrIQ offers maximum flexibility. You don't need to format or rename your Excel columns before uploading. The platform dynamically adapts to your data configuration through two powerful ingestion modes:

🔹 Mode 1: Dynamic Excel Upload Workflow

  • Project Metadata: Start by providing a concise project description (strictly under 10 words) to prime the AI core.
  • Define Scope: Enter the exact number of target columns you want to analyze from your file.
  • Dynamic Context Mapping: The interface will instantly generate a Dropdown Selector and a Context Field for each column. Simply map the original column name, and quickly describe the product and question nature to guide the AI's understanding.

🔹 Mode 2: Direct Copy & Paste (No File Required)

  • Raw Verbatims Field: If you don't have an organized file, simply copy the raw data column from any source (excluding the header) and paste it directly into the provided large text field.
  • Contextual Grounding: Input your project intent and specific question context alongside the pasted text so the AI can build a highly accurate custom Codebook.

How the AI Engine Works (Understanding the Output)

The engine processes data in two main architectural phases to guarantee statistical integrity:

1

Building the Unified Codebook

The AI reviews a representative sample across all uploaded columns to construct a single, standardized project-wide Codebook. This consolidates all potential customer themes, concepts, and ideas (e.g., Price, Quality, Packaging).

2

Response Tagging

The AI processes each consumer response individually, mapping it against the unified project Codebook, and extracts the corresponding Final IDs and Final Text into responsive visual badges.

Frequently Asked Questions (FAQs)

Q: Is my uploaded data secure and private?

Yes, absolutely. All datasets are processed through secure corporate APIs adhering to industry-standard encryption practices. Data access is strictly restricted to authorized project owners within your NarrIQ workspace.

Q: Does the system support multiple open-ended columns simultaneously?

Yes. The platform seamlessly handles Excel files with multiple open-ended textual columns. You can map, configure contexts, and toggle metrics for each column dynamic through the project dashboard.

Q: Can I manually review and edit the AI-generated codes?

Yes, this is a core feature. NarrIQ operates on a "Human-in-the-Loop" architecture. You can click and edit any code name directly in the dynamic table, and the changes will instantaneously sync with the database and update all corresponding verbatims across the screen.

Q: What export formats are supported?

Currently, the platform strictly supports exporting processed results back into fully populated Excel (.xlsx) files for clean data merging. However, our architecture is modular, and custom formats (such as SPSS .sav or CSV) can be seamlessly integrated upon enterprise client request.

Q: How long does the AI analysis take?

Processing time varies based on dataset volume and response complexity. Standard files are processed within hours. For massive enterprise-grade datasets, processing runs asynchronously in the background so you can close the page and return later.

Q: Can I run multiple text analysis jobs at the same time?

Yes. The backend utilizes asynchronous worker queues to handle multiple processing jobs concurrently, ensuring zero downtime and no user interface blocking.

Q: Is the coding pipeline fully autonomous?

No, it is AI-assisted. The engine does the heavy lifting by analyzing and structuring 90%+ of the data, leaving final quality assurance, naming refinements, and strategic alignment entirely under human supervision.
